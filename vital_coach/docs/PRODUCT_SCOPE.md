# Product Scope (v1)

**Vital Coach** is a private, single-user men's health & sexual wellbeing companion with a calm “Today-first” experience.

## MVP outcomes

- One clear **Today** priority + 2–5 next steps.
- Easy logging for core routines (Kegel, pump protocols, recovery resets) with **safety language**.
- A **Coach** screen that explains data and plans (local stub first, Edge Function later).
- RU/EN UI support.
- Privacy-first: discreet UI, no explicit notifications, vault concept.

## MVP modules

- Onboarding (goals, language, privacy posture)
- Today (rule-engine output stub → real engine later)
- Track (sessions + metrics)
- Coach (threads, messages, sources)
- Labs (upload + parse pipeline stub)
- Learning library (curated items + progress stub)

## Non-goals (MVP)

- Full medical diagnosis, guaranteed outcomes, or emergency medicine.
- Stereotype-based relationship compatibility engines.
- “Internet-scale” content scraping without review/tier tagging.
